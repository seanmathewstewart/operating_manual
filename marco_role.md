Operations Analyst

Operations Analyst
- Application level reporting - CRM and Hubspot
- Front-end Power BI reporting
- Ad-hoc analysis
- Assists (but doens't own) Data Quality and Data Governance. Such as Contact and Account deduplication.
- Assists (but doesn't own) data enrichment via data vendors. Such as RingLead.

Not part of the job: 
- Data enrichment
- Managing data vendors
- Operational tasks (for example, get event visits in to crm, xyz is wrong in crm)


Career Growth
- Optimization Partner - If Marco want's to go more towards operations
- Data Analyst - If Marco want's to learn some coding and combine some more technical skills with business skills
- Finance Analysts - If Marco want's to expand his eduction into financial processes
